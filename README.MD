1> INSTALL DEPENDENCY: 
        >>> pip3 install -r /path/to/requirements.txt

        >>> db.create_all()

4> RUN REDIS LOCALLY OR USE REDIS LAB
5> run shell script by:
        >>> chmod a+rx my-script.sh
